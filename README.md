# Меблерія 🛋️

**Сучасний інтернет-магазин меблів** з повною інтеграцією API. Проєкт реалізований з акцентом на продуктивність, адаптивність та зручний користувацький інтерфейс.

## 📌 Про проєкт

**Задача:** Створити багатофункціональний Landing Page меблевого магазину з інтеграцією бекенду для динамічного відображення товарів та обробки замовлень.

### Основні можливості:
* **Каталог:** Повна фільтрація товарів та пагінація.
* **Інтерактивність:** Слайдери для відгуків, акордеони для FAQ, модальні вікна.
* **Форми:** Валідація та відправка замовлень через API.
* **Оптимізація:** loading для зображень та модульна архітектура JS/CSS.

## 🔌 API Інтеграція

Документація бекенду: [Furniture Store API](https://furniture-store-v2.b.goit.study/api-docs/)

**Доступні ендпоінти:**
- `GET /categories` — отримання списку категорій.
- `GET /furnitures` — список товарів з підтримкою пагінації.
- `GET /furniture/{id}` — детальна інформація про конкретний товар.
- `GET /feedbacks` — відгуки покупців.
- `POST /orders` — оформлення замовлення.
